# CallRemind

A simple call reminder web application with Tinder-style swipe cards. For Education Purpose.

## Features
- Add new call reminders with image, name, hometown, purpose and category
- Tinder-like swipe functionality (left = dismiss, right = snooze)
- Filter reminders by category (Emergency, Urgent, Important, No Rush)
- Persistent storage using LocalStorage
- Responsive and modern UI

## What I Learned in JavaScript

- Working with LocalStorage to persist data across page reloads
- Dynamic DOM manipulation and rendering card stacks
- Implementing drag and touch swipe gestures
- Handling form submission and modal interactions
- Managing state and re-rendering UI efficiently
- Event handling for mouse and touch events

This project was built for educational purposes to practice vanilla JavaScript, HTML and CSS.
